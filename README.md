# baby Gole

baby Gole คือเว็บแอปสำหรับผู้ปกครองที่ต้องการดูแลลูกน้อยอย่างเข้าใจง่ายผ่านข้อมูลพัฒนาการ วัคซีน โภชนาการ สมุดพก และชุมชนถาม-ตอบ

## คุณสมบัติ
- หน้า Landing / Auth แบบสวยงาม
- ระบบสมัครสมาชิกและเข้าสู่ระบบ
- หน้าพัฒนาการ วัคซีน โภชนาการ สมุดพก ความรู้ และ AI วิเคราะห์
- ชุมชนถาม-ตอบและระบบแจ้งเตือน
- หน้า Admin Dashboard สำหรับจัดการผู้ใช้และกระทู้

## วิธีเปิดใช้งาน
เปิดไฟล์ [ .vscode/index.html ](.vscode/index.html) ในเบราว์เซอร์ หรือรันเซิร์ฟเวอร์ท้องถิ่น:

```bash
python -m http.server 8000
```

จากนั้นเปิด:

```text
http://127.0.0.1:8000/.vscode/index.html
```

## โครงสร้างหลัก
- [.vscode/index.html](.vscode/index.html) - หน้าเว็บหลัก
- [.vscode/assets](.vscode/assets) - ไฟล์รูปภาพประกอบ
- [.vscode/launch.json](.vscode/launch.json) - การกำหนดรันใน VS Code
